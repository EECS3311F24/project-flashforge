## Requirements
- **Node.js**: 14.x or later
- **npm**
- **Next.js**: 14.x

## Running the Project
1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:  
   In the `~/project-flashforge/doc/sprint1/Website` directory, run:

   ```bash
   npm run dev
   ```
---

Access the site at http://localhost:3000
